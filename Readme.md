# SPINX

> AI-powered developer networking platform built around shared technical experience.

## Features

- Resume Upload
- Skill Extraction
- Intelligent Matching
- Real-time Chat
- Resume Re-matching
- Explainable Recommendations

## Tech Stack

### Frontend

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand
- Framer Motion

### Backend

- Express.js
- TypeScript
- MongoDB
- JWT
- Socket.IO

### Infrastructure

- Docker
- MongoDB
- Redis
- AWS (Future)

## Getting Started

### Client

```bash
cd client
pnpm install
pnpm dev
```

### Server

```bash
cd server
pnpm install
pnpm dev
```
